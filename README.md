# weather_app

Get today&#x27;s and forcast weather data with the help of this beautiful app built using flutter.\
(*Flutter v1.9.1+hotfix.5 or higher required*)

## APIs used

- [Current weather data](https://openweathermap.org/current)
- [5 day weather forecast](https://openweathermap.org/forecast5)

Free OpenWeatherMap API key has been used in this project.\
If you wish to use your own, edit the "appId" variable in lib/utilities.dart.
